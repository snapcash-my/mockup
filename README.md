# SnapCash - Decentralized ATM Network

A comprehensive mobile app mockup showcase for SnapCash, a decentralized ATM network that connects users with local merchants for cash withdrawal services.

## 🚀 Project Overview

This project contains interactive HTML mockups for both the SnapCash User App and Merchant App, along with a professional landing page. The mockups demonstrate the complete user journey from finding nearby merchants to completing cash transactions.

## 📱 App Mockups

### User App (`user-app-mockup.html`)
- **Onboarding**: Welcome screen with app introduction
- **Login/Authentication**: Multiple sign-in options (Google, Apple, Email)
- **Merchant Discovery**: Browse nearby merchants with map/list views
- **Cash Request**: Request specific amounts from merchants
- **Payment Processing**: Secure DuitNow QR payments
- **Transaction History**: View past transactions and receipts
- **User Profile**: Account management and settings

### Merchant App (`merchant-app-mockup.html`)
- **Authentication**: Secure merchant login and registration
- **Dashboard**: Overview of requests and earnings
- **Transaction Management**: Review and approve cash requests
- **QR Payment Display**: Generate payment codes for users
- **Earnings Analytics**: Track revenue and performance metrics
- **Profile Management**: Business information and settings
- **ezyLite POS Integration**: Built-in POS system

## 🌐 Landing Page (`index.html`)

A professional marketing landing page featuring:
- Hero section with compelling messaging
- Feature highlights and benefits
- Interactive app demonstrations
- Call-to-action buttons linking to app mockups
- Responsive design for all devices

## 🛠️ Technology Stack

- **Frontend**: HTML5, Tailwind CSS, JavaScript
- **Icons**: Font Awesome 6.4.0
- **Images**: Picsum Photos for placeholder images
- **Deployment**: Cloudflare Pages (static hosting)

## 🚀 Deployment

### Cloudflare Pages Deployment

1. **Connect Repository**
   - Push this repository to GitHub/GitLab/Bitbucket
   - Connect the repository to Cloudflare Pages

2. **Build Settings**
   - **Build command**: Not required (static site)
   - **Build output directory**: `/`
   - **Root directory**: `/`

3. **Environment Variables**
   - No environment variables required

4. **Deploy**
   - Cloudflare Pages will automatically build and deploy
   - Your site will be available at `https://your-project.pages.dev`

### Local Development

1. Clone the repository
2. Open `index.html` in your browser to view the landing page
3. Navigate to app mockups using the links provided
